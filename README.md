Appointment Planner

A React application for managing contacts and scheduling appointments. Users can add contacts, create appointments linked to them, and view everything in a clean, organized interface.

Features
Add new contacts
Prevent duplicate contact names
Create appointments linked to existing contacts
Select contacts from a dropdown when scheduling
View contacts and appointments in reusable UI cards
Fully reactive interface built with React Hooks
Tech Stack
React (Functional Components)
React Hooks (useState)
React Router
JavaScript (ES6+)
HTML & CSS
