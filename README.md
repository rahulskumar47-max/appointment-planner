# 📅 Appointment Planner

A React application that allows users to manage contacts and schedule appointments. Built using functional components, React Hooks, and React Router.

---

## ✨ Features

- ➕ Add new contacts
- 🚫 Prevent duplicate contact names
- 📅 Create appointments linked to contacts
- 🧑‍🤝‍🧑 Select contacts from a dropdown
- 🧾 View all contacts and appointments in reusable cards
- 🔄 Fully reactive UI using React Hooks

---

## 🧠 Tech Stack

- React (Functional Components)
- React Hooks (useState)
- React Router
- JavaScript (ES6+)
- HTML/CSS
